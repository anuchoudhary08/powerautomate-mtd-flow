## ⚙️ Workflow Logic & Processing

This system is designed to automate employee reporting and deliver structured performance insights to managers using Power Automate.

### 🔹 Step 1: Trigger (Form Submission)
- The workflow is triggered automatically when an employee submits the Microsoft Form  
- Captures daily work details including employee name, date, and task  



### 🔹 Step 2: Data Extraction & Processing
- Retrieves form response using Power Automate  
- Extracts key fields such as employee details and manager email  
- Applies **Month-To-Date (MTD) logic** to organize reporting data  



### 🔹 Step 3: Business Logic Implementation
- Identifies the respective manager based on submitted data  
- Structures the response into a readable format  
- Ensures consistent and accurate reporting  



### 🔹 Step 4: Automated Email Notification
- Sends a structured email to the manager  
- Includes employee work details and reporting summary  
- Eliminates need for manual follow-up  



### 🔹 Step 5: Data Storage for Analytics
- Stores response data in a structured format (Excel / SharePoint)  
- Acts as a data source for Power BI  



### 🔹 Step 6: Visualization & Insights
- Power BI dashboard is connected to stored data  
- Provides performance tracking, trends, and MTD analysis  



## 🎯 Outcome

- Automated end-to-end reporting workflow  
- Reduced manual effort and errors  
- Improved visibility of employee performance  
- Enabled data-driven decision-making through dashboards
